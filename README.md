# 📝 To-Do List

Uma aplicação simples de lista de tarefas desenvolvida com **HTML, CSS e JavaScript puro**, com foco em organização, usabilidade e persistência de dados no navegador.

---

## 🚀 Funcionalidades

- ✅ Adicionar tarefas com descrição  
- 🎯 Definir prioridade (Alta, Média ou Baixa)  
- 📅 Inserir data e hora para execução  
- ✔️ Marcar tarefas como concluídas  
- ✏️ Editar tarefas existentes  
- 🗑️ Excluir tarefas  
- 💾 Armazenamento automático com `localStorage`  
- 🔍 Filtro por:
  - Todas
  - Prioridade
  - Concluídas  
- 🔄 Ordenação automática por prioridade  
- 🎨 Estilização condicional (cores diferentes por prioridade)  
- 🌙 Modo escuro com alternância dinâmica
